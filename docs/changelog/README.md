# Changelog Docs

Canonical home for detailed changelog source entries.

## Ownership
- Primary: Release Manager
- Update cadence: every major release PR merge

## Backward compatibility
- `CHANGELOG.md` remains the top-level generated/indexed entry point.
- Author or update detailed records here first, then sync/aggregate into `CHANGELOG.md`.

## Contents
- Use `entry-template.md` when adding new changelog records.

# Release Docs

Canonical release documentation and release-note source entries.

## Ownership
- Primary: Release Manager
- Update cadence: at release candidate and general availability checkpoints

## Backward compatibility
- `docs/release-notes.md` remains the generated/indexed summary feed.
- Release entries in this folder are the detailed source of truth.

## Contents
- `release-note-entry-template.md` for each release-note entry.
- `manifest/current.json` for machine-readable current release metadata.
